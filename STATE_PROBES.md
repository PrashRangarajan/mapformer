# Spatial-state probes: position decode + goal-distance (s0)

Reads the explicit spatial state from each model:

- **Level15:** ``model.last_theta_hat`` (InEKF-corrected angles).

- **Vanilla:** ``theta_path`` recomputed on the fly (uncorrected cumsum of action deltas).

- **TEMFaithful:** ``g`` (structural code) via manual step rollout.


Not the post-LayerNorm hidden state used by ``probe_goal_distance.py``.


## Position decode (displacement from start)

Target: agent (dx, dy) mod 64. Encoded as cos/sin so the head can
predict on the circle. 'MAE cells' = average torus-distance between

predicted and true displacement. Random-guess baseline ~32 cells.
