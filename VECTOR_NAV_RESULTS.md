# Vector navigation probe — Tolman cognitive-map test

Tests whether prediction-trained representations support navigation
to ARBITRARY landmarks without any goal-directed training. The model
has only seen random walks; we probe whether its hidden state encodes
the spatial relationship between current position and any landmark.

**Probe:** given (hidden_t, target_landmark_id), predict the BFS-optimal
next action to reach that landmark. Linear head + learnable landmark
embedding. Frozen backbone, all gradient flows through probe only.

**Key split:** test accuracy separated by whether the target landmark
was VISITED during the random walk so far. If accuracy is similar for
visited and unvisited landmarks, the model has a TRUE cognitive map —
it knows landmark positions even before encountering them.

Multi-seed (n=3 prediction-trained lm200 checkpoints). Held-out env (seed=1000).

## Interpretation

- If RoPE near chance: no cognitive map; the representation has no spatial info
