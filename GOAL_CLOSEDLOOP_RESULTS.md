## Interpretation

- **Match-acc** is open-loop: feed the expert trajectory, ask what action the model would pick. Doesn't compound errors.
- **Closed-loop success** is end-to-end: model picks actions, env executes, agent must actually reach goal. Compounds errors.
- If closed-loop ≈ match-acc: cognitive map supports re-planning on the fly. Model can recover from its own mistakes mid-trajectory.
- If closed-loop ≪ match-acc: BC distribution-shift; per-step errors compound and agent gets lost.
