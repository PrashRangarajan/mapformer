# Active-inference probe: is the cognitive map usable for goal-directed action?

Frozen single-env lm200 checkpoints (random-walk training, NO goal
conditioning, NO policy training). At each navigate step, for each
candidate action, score by max log p(goal_token) over a `horizon`-step
rollout. Pick argmax. Closed-loop execution — model selects action,
env executes, next obs feeds back.

Setup: T_explore=64 random walk → T_navigate=32 active-inference. Goal
= random landmark. 50 episodes/variant. Single seed (s0).

**Theoretical caveat:** horizon=1 is structurally limited — the model
predicts only the *next* observation, so log p(goal_token) is only
informative when adjacent to the goal. Multi-step horizons (4, 8) use
Dreamer-style argmax-obs rollouts to extrapolate further.
