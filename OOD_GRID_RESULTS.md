## Interpretation

- IID should be near-ceiling for MapFormer family; RoPE collapses.
- OOD-d tests denser-smaller-shorter generalization (32×32, p_empty=0.2, T=64). Should be relatively easy.
- OOD-s tests sparser-larger-longer (128×128, p_empty=0.8, T=512). The hardest setting: cumulative angle grows further, fewer informative observations, more drift.
- The omega-rescaling trick (×2 for 32×32, ×0.5 for 128×128) should make MapFormer variants scale-robust. Without it, the model assumes a 64×64 world during path integration.
